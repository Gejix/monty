# 0x19. C - Stacks, Queues - LIFO, FIFO
## monty program description

This Project features a custom Matlab ByteCode interpreter. Project is written in C.

## Compilation 

    gcc -Wall -Werror -Wextra -pedantic *.c -o monty

## Usage

    ./monty byte_file.m
